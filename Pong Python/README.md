# 🏓 Pong Clásico con Python & Pygame

Este es un clon del legendario Pong, el abuelo de los videojuegos, desarrollado íntegramente en Python. Es un proyecto ideal para ver cómo funciona la lógica de colisiones, el manejo de eventos de teclado y el renderizado de gráficos en tiempo real. 🕹️

## 🚀 Sobre Pygame

Para dar vida a este juego, he utilizado Pygame, una librería de Python diseñada específicamente para crear videojuegos en 2D. Es una herramienta súper versátil que permite:

- ☑️  Gestionar gráficos y sonidos de forma sencilla.
- ☑️  Controlar los FPS (frames por segundo) para que el juego corra fluido.
- ☑️  Manejar la física básica de rebotes y movimiento de objetos.

## 🎮 Cómo jugar:

El duelo es para dos jugadores en el mismo teclado:

- ☑️ Jugador Izquierdo: Usa las teclas W (arriba) y S (abajo).
- ☑️ Jugador Derecho: Usa las Flechas de Dirección (arriba y abajo).
