Fichero de configuración para FASTFETCH

- El ejemplo incluye información acerca de la IP pública de la conexión a internet

- Para generar archivo de configuración por defecto para modificar: 
  fastfetch --gen-config (se genera en ~/.config/fastfetch/config.jsonc)

- Para ver diferentes configuraciones creados por usuarios de FASTFETCH:
  https://github.com/topics/fastfetch-presets
  (descargar los presets y mover y renombrar a ~/.config/fastfetch/config.jsonc)

NOTA:
Siempre es conveniente copiar el archivo de configuración por defecto a un respaldo (ej.: config.bak) antes de realizar cambios.
