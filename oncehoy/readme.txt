**Generador Diario de Números de Lotería**

Este script en bash proporciona una combinación única de números de lotería cada día, diseñada para fines de entretenimiento y esparcimiento.

**Características principales:**
- Genera cifra de 5 dígitos al azar. 
- Combinación diferente cada vez que se invoca.
- Formato claro y fácil de leer

**Uso:**
./oncehoy.sh

**Nota:** Este es un proyecto recreativo. Los números se generan aleatoriamente y no garantizan resultados en sorteos reales. Utilícelo responsablemente y como fuente de diversión.

Ideal para grupos de oficina, amigos o como elemento de entretenimiento casual.
